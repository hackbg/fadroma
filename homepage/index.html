Supported platforms

Secret Network is an open-source, permissionless blockchain based on original privacy research and papers first written at MIT in 2015, including "Decentralizing Privacy", now one of the most influential papers in the blockchain space with 2,000 academic citations.

Provided capabilities

Derive contract

Procedural macro for generating smart contract boilerplate

Reproducible builder

Perform production builds of your contracts in a Docker container

Scriptable deployments

Manage hundreds of heterogeneous contracts using TypeScript without a compilation step

Automatic devnet management

Spawn local blockchain nodes for development and testing

Transaction receipts

Keep track of uploads, instantiations, and configuration

Contract clients

Define contract-specific client classes to call contract methods perform off-chain logic in TypeScript

Ensemble testing

Test multiple interoperating smart contracts using a Rust API

Executable specifications

Test your smart contracts and off-chain logic from TypeScript without the overhead of testing frameworks

Environment-based configuration

Configure your workflow using environment variables and/or a .env file

256-bit arithmetic

256-bit integer support in Cosmos for easy EVM interop

On-chain checksum generation

Generate SHA256 checksums for arbitrary data from your contract

Pseudo-random number generator

ChaCha-based PRNG

Contract ownership

Viewing keys
