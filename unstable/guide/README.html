Overview #

Fadroma is a full-stack application framework for the CosmWasm ecosystem.

This guide will show you how to set up a Fadroma project.

The Rust part #

Fadroma Engine is a collection of Rust libraries for developing smart contracts.

Fadroma Engine includes Fadroma Derive, a collection of procedural macros for cleaner implementation of smart contract internals, and Fadroma Ensemble, a library for integration testing of multiple contracts, as well as a range of other useful bits and pieces.

The TypeScript part #

Fadroma Client is a library for interfacing with smart contracts from JavaScript or TypeScript.

Fadroma Client is leveraged by Fadroma Ops, a library for implementing your custom deployment and operations workflow - from local development to mainnet deployment.

Fadroma Ops includes Fadroma Mocknet, a simulated environment for fast full-stack testing of your production builds.
