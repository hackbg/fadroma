constructor

  • new RestClient(apiUrl: string, broadcastMode?: SecretJS.BroadcastMode, seed?: Uint8Array): SecretJS.RestClient
    • Defined in node_modules/.pnpm/secretjs@0.17.5/node_modules/secretjs/types/restclient.d.ts:208

    Creates a new client to interact with a Cosmos SDK light client daemon. This class tries to be a direct mapping onto the API. Some basic decoding and normalizatin is done but things like caching are done at a higher level.

    When building apps, you should not need to use this class directly. If you do, this indicates a missing feature in higher level components. Feel free to raise an issue in this case.

    Parameters

    • apiUrl: string

      The URL of a Cosmos SDK light client daemon API (sometimes called REST server or REST API)

    • Optional broadcastMode: SecretJS.BroadcastMode

      Defines at which point of the transaction processing the postTx method (i.e. transaction broadcasting) returns

    • Optional seed: Uint8Array

      The seed used to generate sender TX encryption key. If empty will generate random new one

    Returns SecretJS.RestClient
