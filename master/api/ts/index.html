Fadroma aims to take the CosmWasm Compute module and spin it into a fully integrated app platform.

This repository contains assorted Rust libraries for smart contracts and a TypeScript system for building, deploying, and interacting with them.

Check out our example project to see how to build your dApp with Fadroma.
